# Contributing to Control Narrative Analysis

Thank you for your interest in contributing to this project!

## How to Contribute

1. **Bug Reports**: If you find a bug, please open an issue.
2. **Feature Requests**: If you have an idea for a new feature, please open an issue to discuss it.
3. **Pull Requests**: 
    - Fork the repository.
    - Create a new branch for your feature or bug fix.
    - Submit a pull request with a clear description of your changes.

## Development Setup

1. Install dependencies: `pip install -r ml_prototype/requirements.txt`
2. Start the server: `python app.py`

## Code Style

Please follow PEP 8 for Python code and standard web development best practices for HTML/CSS/JS.
